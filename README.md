*Este proyecto contiene mi entorno gráfico completo basado en **BSPWM** para Kali Linux.*

![menu](hacker/hacker.jpeg)

## Clonar repositorio
```bash
- git clone https://github.com/espinalclark/sddm-themes.git
- cd sddm-themes
- chmod +x setup.sh
- ./setup.sh
```
## Instalacion

**Arch Linux**

```bash
- chmod +x setup-arch.sh 
- ./setup-arch.sh 

```
**Kali Linux**

```bash
- chmod +x setup-kali.sh 
- ./setup-kali.sh


```
